# Transcribe API Documentation

Базовый URL: `http://localhost:3000` (или хост, на котором запущен сервис)

## Эндпоинты

### 1. Проверка статуса сервиса (Healthcheck)
Позволяет убедиться, что API работает.

* **URL:** `/health`
* **Метод:** `GET`
* **Ответ (200 OK):**
```json
{
  "status": "ok",
  "service": "transcribe-api"
}
```

---

### 2. Загрузка файла на транскрибацию
Принимает аудиофайл, сохраняет его и ставит в очередь RabbitMQ на обработку.

* **URL:** `/transcribe`
* **Метод:** `POST`
* **Query-параметры:**
  * `chat_id` (строка, **обязательно**) — идентификатор чата (например, из Telegram).
  * `message_id` (строка, *необязательно*) — идентификатор сообщения.
* **Тело запроса:** `multipart/form-data`
  * `file` (файл, **обязательно**) — аудиофайл. Поддерживаемые форматы: `ogg`, `mp3`, `wav`, `webm`, `m4a`. Максимальный размер: 500 MB.
* **Успешный ответ (200 OK):**
```json
{
  "task_id": "4d1db26d-2d4e-4f3a-96e5-4f3183592161",
  "status": "queued",
  "file_path": "/tmp/transcribe/4d1db26d-2d4e-4f3a-96e5-4f3183592161_audio.ogg"
}
```
* **Ответы с ошибками:**
  * `400 Bad Request`: Неверный формат файла, отсутствует файл или отсутствуют обязательные query-параметры (например, `chat_id`).
  * `500 Internal Server Error`: Ошибка при сохранении файла.
  * `503 Service Unavailable`: Ошибка отправки задачи в RabbitMQ.

---

### 3. Получение результата
Возвращает JSON с результатами транскрибации (от whisper.cpp) по идентификатору задачи.

* **URL:** `/result/{task_id}`
* **Метод:** `GET`
* **Параметры пути:**
  * `task_id` (строка) — ID задачи, полученный при загрузке файла.
* **Успешный ответ (200 OK):**
```json
{
  "transcription": [
    {
      "timestamps": {"from": "00:00:00,000", "to": "00:00:02,000"},
      "offsets": {"from": 0, "to": 2000},
      "text": " Привет, это тестовая запись."
    }
  ]
}
```
*(Точная структура зависит от вывода `whisper-cli -oj`)*
* **В процессе или не найдено (404 Not Found):**
```json
{
  "status": "processing_or_not_found"
}
```
*(Возвращается, если файл с результатом еще не создан или задача не существует).*
* **Ошибка (400 Bad Request):**
  * Если `task_id` содержит недопустимые символы.

---

## Примеры использования (cURL)

**1. Загрузка файла:**
```bash
curl -X POST "http://localhost:3000/transcribe?chat_id=12345&message_id=678" \
     -F "file=@/path/to/your/audio.ogg"
```

**2. Получение результата:**
```bash
curl -X GET "http://localhost:3000/result/4d1db26d-2d4e-4f3a-96e5-4f3183592161"
```
